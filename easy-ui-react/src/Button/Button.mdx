# Button

A `<Button />` enables users to take an action or transition to other pages. They are designed with multiple styles to address various needs, and they are ideal for highlighting an action for the user to take in order to move forward in a flow.

**Note**: Each `<Button />` variant support a subset of colors from the `color` property.

## Custom Brand Colors

Use `color="brand"` along with overrides in a wrapped `<ThemeProvider>` to render custom brand colors.

This can be used in applications to create buttons for custom brands.

```jsx
function NortonButton(buttonProps: Omit<ButtonProps, "color">) {
  return (
    <ThemeProvider
      theme={() => ({
        "button.brand.filled.font.color":
          "var(--ezui-theme-light-color-neutral-900)",
        "button.brand.filled.active.font.color":
          "var(--ezui-theme-light-color-neutral-900)",
        "button.brand.resting.color": "var(--ezui-theme-light-color-warning-500)",
        "button.brand.hover.focus.color":
          "var(--ezui-theme-light-color-warning-600)",
        "button.brand.active.color": "var(--ezui-theme-light-color-warning-700)",
      })}
    >
      <Button {...buttonProps} color="brand" />
    </ThemeProvider>
  );
}
```
