# CodeBlock

A `<CodeBlock />` is used to improve readability of embedded code samples with syntax highlighting and automatic line numbering.

Use the `code` and `language` props to control the contents of the `<CodeBlock />`.

<Canvas of={CodeBlockStories.Default} />

<Controls of={CodeBlockStories.Default} include={["language"]} />

## Line Numbers

A `<CodeBlock />` can show line numbers using `showLineNumbers`.

<Canvas of={CodeBlockStories.LineNumbers} />

<Controls of={CodeBlockStories.LineNumbers} include={["showLineNumbers"]} />

## Max Lines

A `<CodeBlock />` can limit its height to a maximum number of lines using the `maxLines` prop.

<Controls of={CodeBlockStories.MaxLines} include={["maxLines"]} />
