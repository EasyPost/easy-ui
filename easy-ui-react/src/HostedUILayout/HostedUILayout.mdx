# HostedUILayout

`HostedUILayout` defines a header and main content area for a HostedUI product page.

`HostedUILayout` is a compound component consisting of `HostedUILayout`, `HostedUILayout.Header`, and `HostedUILayout.Content`.

`HostedUILayout` also provides several components for constructing the header, including `HostedUILayout.LogoContainer` and `HostedUILayout.Actions`.

<Canvas of={HostedUILayoutStories.StandardContent} />

## With MultipageSection

`HostedUILayout` can be combined with `MultipageSection` to render a multipage navigational container.
