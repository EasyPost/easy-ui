# SplitToggleCard

A `<SplitToggleCard />` is an interactive binary control with related content.

A `<SplitToggleCard />` can be selected using `defaultSelected` (uncontrolled).

## Selected

A `<SplitToggleCard />` can be selected using `isSelected` (controlled).

<Controls of={SplitToggleCardStories.Selected} include={["isSelected"]} />

## Disabled

A `<SplitToggleCard />` can be disabled using `isDisabled`.

  of={SplitToggleCardStories.Disabled}
