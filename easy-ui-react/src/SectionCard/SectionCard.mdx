# SectionCard

A `SectionCard` is a styled container for grouping section content on a page.

`SectionCard` is designed for composability. It can be deconstructed and reassembled as needed.

<Canvas of={SectionCardStories.WithTabPanels} />

## Basic

A basic `SectionCard` renders a stylized `Card` for page sections.

<Canvas of={SectionCardStories.Basic} />

## With Header

Use a `SectionCard.Header` to render a header.

`SectionCard.Header` supports `SectionCard.Title`, `SectionCard.Controls`, and `SectionCard.Actions`.

<Canvas of={SectionCardStories.WithHeader} />

## With Tabs

Use the `SectionCard.Tabs` helper to wrap `TabPanels` or `TabNav`.
