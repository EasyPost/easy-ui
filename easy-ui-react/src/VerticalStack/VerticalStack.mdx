# VerticalStack

A `<VerticalStack />` is used to display children vertically based on CSS flexbox.

<Canvas of={VerticalStackStories.Default} />

## Gap

A `<VerticalStack />` can have a custom gap between elements using Easy UI's space scale.

<Canvas of={VerticalStackStories.Gap} />

<Controls of={VerticalStackStories.Gap} include={["gap"]} />

`gap` can also be set on a per-breakpoint basis:

```tsx
<VerticalStack gap={{ xs: "2", sm: "3", md: "4", lg: "5", xl: "6" }}>
  <Content />
</VerticalStack>
```
