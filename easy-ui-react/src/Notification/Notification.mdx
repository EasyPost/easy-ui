# Notification

The stories below simulate how a notification of type `alert` or `toast` is triggered. In practice, one would use Easy UI's
`useNotification` hook, which returns an object on which the following methods can be invoked: `showSuccessToast`, `showPromotionalToast`,
`showNeutralToast`, `showWarningToast`, `showErrorToast`, `showSuccessAlert`, `showPromotionalAlert`, `showNeutralAlert`, `showWarningAlert`,
and `showErrorAlert`. See [notification documentation specs](https://github.com/EasyPost/easy-ui/blob/main/documentation/specs/Notification.md#example-usage) for more
information on example usage.
