## Scrolls on Max Items

When a `<Menu />` has more than five items the menu scrolls to only show those items. The number can be configured with the `maxItemsUntilScroll` property. Set the value to `Infinity` to disable scrolling.

<Canvas of={MenuStories.ScrollsOnMaxItems} />

<Controls

  include={["maxItemsUntilScroll"]}
/>

## Width Options

By default, a `<Menu />` is the width of its trigger element and stretches to the length of its content.

<Canvas of={MenuStories.WithLongItems} />

By setting `width="fit-content"` the menu will fit to the width of its content regardless of the width of its trigger.

<Canvas of={MenuStories.MatchContentWidth} />

The `width` of the `<Menu.Overlay />` can also be set to any arbitrary value.

<Canvas of={MenuStories.DefinedWidth} />

The `width` of the `<Menu.Overlay />` can also be set to any arbitrary value based on breakpoints.

<Canvas of={MenuStories.DefinedResponsiveWidth} />

## Item Types

A `<Menu />` can include links by using the `href` property, and it can include disabled menu items by using the `disabledKeys` property on the `Menu.Overlay`.

<Canvas of={MenuStories.MixedItemTypes} />

## Custom Placement

A `<Menu />` can be placed on different edges of its trigger element. See the controls for valid values. Ensure any deviation from the default value is coordinated with design to prevent unexpected user experience.
