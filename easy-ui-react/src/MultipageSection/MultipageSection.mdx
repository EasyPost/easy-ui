# MultipageSection

`MultipageSection` represents a section for multipage navigation and a content area that is preceded by a branded header.

`MultipageSection` is a compound component consisting of `MultipageSection`, `MultipageSection.Container`, `MultipageSection.BrandHeader`, `MultipageSection.Sidebar`, and `MultipageSection.Content`.

`MultipageSection` also provides components for building the sidebar navigation with `MultipageSection.SidebarNav`.
