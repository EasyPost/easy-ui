# Card

Cards are styled containers that group related content.

## Outlined

By default cards are rendered with a border. This is the same as `variant="outlined"`.

<Canvas of={CardStories.Outlined} />

<Controls of={CardStories.Outlined} />

## Solid

Cards can be borderless using `variant="solid"`.

<Canvas of={CardStories.Solid} />

<Controls of={CardStories.Solid} />

## Flagged

Cards can be flagged using `variant="flagged"`. Use the `status` prop to adjust the flag style.

<Canvas of={CardStories.Flagged} />

<Controls of={CardStories.Flagged} />

## Composition

Cards are composable. Use `<Card.Container />` and `<Card.Area />` to create cards with complex content.

`<Card.Container />` is the outer element of a card containing its border.

`<Card.Area />` is the inner element of a card containing its padding and background.

To create a card with two columns, for instance, use `<Card.Container />` and `<Card.Area />`.
