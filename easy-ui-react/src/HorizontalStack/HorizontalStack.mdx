# HorizontalStack

A `<HorizontalStack />` is used to display children horizontally based on CSS flexbox.

<Canvas of={HorizontalStackStories.Default} />

## Gap

A `<HorizontalStack />` can have a custom gap between elements using Easy UI's space scale.

<Canvas of={HorizontalStackStories.Gap} />

<Controls of={HorizontalStackStories.Gap} include={["gap"]} />

`gap` can also be set on a per-breakpoint basis:

```tsx
<HorizontalStack gap={{ xs: "2", sm: "3", md: "4", lg: "5", xl: "6" }}>
  <Content />
</HorizontalStack>
```
