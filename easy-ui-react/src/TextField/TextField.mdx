# TextField

The `<TextField />` component allows users to input text on a single line and provides the essentials to be used as a form control element.

## Standard

Simple `<TextField />` with `label`, `placeholder`, and `helperText`.

<Canvas of={TextFieldStories.Standard} />

## Password

Setting `type="password"` yields a password input with a visibility icon button.

<Canvas of={TextFieldStories.Password} />

## Icon

Use `iconAtStart` or `iconAtEnd` to add an icon.

<Canvas of={TextFieldStories.Icon} />

## Small TextField

Set `size="sm"` for a small `<TextField />`.

<Canvas of={TextFieldStories.SmallTextFields} />

## Large TextField

Set `size="lg"` for a large `<TextField />`.

<Canvas of={TextFieldStories.LargeTextFields} />

## Label with emphasis

Use `emphasizedLabel=true` to display a label with emphasis.

<Canvas of={TextFieldStories.LabelWithEmphasis} />

## Error

When `validationState` is set to `invalid`, the `<TextField />` displays with error styles.

<Canvas of={TextFieldStories.Error} />
