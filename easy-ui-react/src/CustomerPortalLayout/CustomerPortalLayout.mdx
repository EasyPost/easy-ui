# CustomerPortalLayout

`CustomerPortalLayout` defines a header and main content area for a CustomerPortal product page.

`CustomerPortalLayout` is a compound component consisting of `CustomerPortalLayout`, `CustomerPortalLayout.Header`, and `CustomerPortalLayout.Content`.

`CustomerPortalLayout` also provides several components for constructing the header, including `CustomerPortalLayout.LogoContainer` and `CustomerPortalLayout.Actions`.

<Canvas of={CustomerPortalLayoutStories.StandardContent} />

## Test Mode

`CustomerPortalLayout` can display a prominent message for indicating test mode with the `mode="test"` prop.

<Canvas of={CustomerPortalLayoutStories.TestMode} />

## Display EasyPost Logo

By default, `CustomerPortalLayout` displays the EasyPost logo. To prevent the logo from displaying, set the `shouldDisplayEasyPostLogo` prop to `false`.

<Canvas of={CustomerPortalLayoutStories.DisplayEasyPostLogo} />

## With MultipageSection

`CustomerPortalLayout` can be combined with `MultipageSection` to render a multipage navigational container.
